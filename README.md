# Shopify Product Scraper

A Python-based scraper to extract product data from Shopify stores.

## Features
- Scrape product title, price, image
- Handle pagination automatically
- Multi-store support
- Export to CSV
- Save to MongoDB

## Tech Stack
- Python (requests)
- MongoDB
- CSV

## How to Run

```bash
pip install -r requirements.txt
python scraper.py
